Welcome to the EchoVR texture editor,

App is now easy and simple to use, click tutorial in top left if you get stuck.

If you have any questions message @he_is_the_cat on discord.

Special thanks to [Exhibitmark](https://github.com/Exhibitmark) for reverse engineering EVR's manifest & package format, and for making [Carnation](https://github.com/Exhibitmark/carnation).

Special thanks to [Goopsie](https://github.com/Goopsie) and [sprockee](https://github.com/thesprockee) for making the [original version](https://github.com/goopsie/evrFileTools) and [updated version of evrFiletools](https://github.com/EchoTools/evrFileTools) respectively, the tools used in this project to extract and replace EVR's assets.

Here is every texture for cosmetics, use these as your example when creating new cosmetics: [Examples.zip](https://github.com/user-attachments/files/25971111/Examples.zip)

[Tutorial](https://youtu.be/iEeCr-4pQJA?si=VpUQGNr38H1gbwG2)
